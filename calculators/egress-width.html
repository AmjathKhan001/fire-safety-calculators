Egress Width Calculator (NFPA 101 / IBC)

Calculates the minimum required total width of all exit paths (doors, stairs, ramps) based on the calculated occupant load.

Egress Component

Factors: 0.2" (stairs), 0.15" (other) for reduced capacity; 0.3" (stairs), 0.2" (other) for standard.

Minimum Width Requirements

  • Minimum door width: 32 inches clear
  • Minimum stair width: 44 inches for > 50 occupants
  • Ramps: Same as corridors (typically 44 inches)
  • Corridors: Minimum 44 inches for > 50 occupants
  • Width measured clear of obstructions
