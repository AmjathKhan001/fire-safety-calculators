Smoke Detector Spacing Calculator (NFPA 72)

Calculates the **maximum allowable spacing** between spot-type smoke detectors based on ceiling height and type, as required by NFPA 72. **Room length and width are not required** for determining maximum spacing.

Fire Damper / Smoke Damper Selection Guide

Occupant Load Calculator (NFPA 101 / IBC)

Calculates the maximum allowed number of occupants based on floor area and the prescribed occupancy factor.

Area used for determining occupant load.
Factors vary by code edition and specific usage. Consult NFPA 101 or IBC.

Egress Width Calculator (NFPA 101 / IBC)

Calculates the minimum required total width of all exit paths (doors, stairs, ramps) based on the calculated occupant load.

Use the result from the Occupant Load Calculator.
Factors: 0.2" (stairs), 0.15" (other) for reduced capacity; 0.3" (stairs), 0.2" (other) for standard.

Emergency Lighting Spacing Calculator

Estimates the maximum fixture spacing based on mounting height, light intensity, and the minimum required illuminance (simplified inverse square law).

Light intensity output from the emergency fixture.
Typically 1.0 ft-c or 10.76 Lux (NFPA 101/IBC).

Hydrant Flow Calculator (NFPA 291)

Calculates the flow rate from an open hydrant outlet based on the measured velocity pressure (pitot pressure) and outlet diameter.

Measured velocity pressure using a pitot gauge.
Internal diameter of the flowing outlet (e.g., 2.5 inches).
Usually 0.90 for a smooth, well-rounded outlet (NFPA 291).

Friction Loss Calculator (Hazen-Williams)

Calculates the pressure drop due to friction in a pipe run using the Hazen-Williams formula.

120 for new steel pipe, 100 for black steel, 140 for new plastic.

Required Fire Flow Calculator (NFPA 1142)

Estimates the minimum required fire flow for a structure based on building area and construction type (NFPA 1142 formula).

Total area of all floors for the largest fire compartment.
Formula: F = 18 * C * sqrt(Area).

Fire Pump Sizing & Head Calculator

Calculates the minimum required pump head (pressure) based on system demands, losses, and available water supply.

Flow required by the sprinkler/hose system (e.g., 500 GPM).
Pressure needed to lift water to the remote point (0.433 PSI/ft).
Pressure supplied to the pump from the city water main or tank.
